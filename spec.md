# Specification

## Summary
**Goal:** Convert the Mount Paradise School website from Hindi to English language across all pages and components.

**Planned changes:**
- Translate all page content (Home, About, Academics, Admissions, Gallery, Contact, Registration) from Hindi to English
- Convert navigation menu items to English (Home, About, Academics, Admissions, Gallery, Contact, Student Registration)
- Update all form labels, placeholders, and validation messages to English
- Translate hero sections, feature cards, CTA buttons, and descriptions to English
- Convert mission, vision, curriculum information, and enrollment process details to English
- Update footer links, contact labels, and all UI text elements to English

**User-visible outcome:** The entire website displays in English, making it accessible to English-speaking visitors while maintaining all existing functionality and design.
